[A, B, C, D, E]というリストから  
{1:A, 2:B, 3:C, 4:D, 5:E}というディクショナリを作る

```
>>> l = ['A', 'B', 'C', 'D', 'E']
>>> l
['A', 'B', 'C', 'D', 'E']
>>> dict([(k, v) for k, v in enumerate(l)])
{0: 'A', 1: 'B', 2: 'C', 3: 'D', 4: 'E'}
>>> 
```

enumerateというものあってリストのインデックスと値を返してくれる
