### mvimコマンドをダウンロード
```bash
$ wget http://repo.or.cz/w/MacVim/KaoriYa.git/blob_plain/HEAD:/src/MacVim/mvim ~/bin/mvim
$ chmod 0755 ~/bin/mvim
```

### aliasを設定
.bashrcなどに追加

```bash
#常に新規タブで開く
alias vim="mvim --remote-tab-silent"
```
